NLP on hair products

A study of how Skala's hair products group on a 2D plane.

Python
Pandas
Bokeh
BeautifulSoup
Scikit-learn
NLP
RegEx
Dimensionality reduction

For this project I wrote two Jupyter notebooks: one that allows you to download ingredient lists from a brand's website, and the other one to study/visualize them by using One-Hot encoding, T-distributed Stochastic Neighbor Embedding (t-SNE) and an interactive Bokeh app.

The Bokeh app is available here below as well. By using the "box select" tool it is possible to select products, to compare their ingredients and see how they group in space as a function of the kind of product.

For example, shampoos clearly clump together in a separate region of the space, but the same cannot be said about the rest of categories.

As far as ingredients are concerned, by selecting two nearby products one can see which common ingredients they have, and the characterizing actives.
